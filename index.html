Une architecture modulaire pensée pour tourner sur du matériel humble, avec une expérience complète.

  • 🧠

    LLM Local via Ollama

    Llama, Mistral, Gemma — choisissez votre modèle. Aucune API externe, aucune latence réseau, zéro frais.

  • 🎙️

    Speech-to-Text Whisper (à venir)

    Transcription vocale locale avec Whisper (modèle open source, aucune donnée envoyée). Tourne entièrement sur votre machine.

  • 🔊

    Text-to-Speech natif (à venir)

    Réponses vocales avec eSpeak. Néron vous répond à voix haute, sans connexion internet.

  • 💾

    Mémoire persistante

    SQLite embarqué. Néron se souvient de vos conversations, préférences et contexte au fil du temps.

  • 📱

    Interface Telegram

    Discutez avec Néron depuis votre téléphone via Telegram. Commandes, conversations, notifications.

  • ⚙️

    Service systemd

    Démarre automatiquement au boot, redémarre en cas de crash. Gérez tout avec make start/stop.

Un seul processus.
Plusieurs agents.

Néron v2 abandonne Docker pour un seul processus Python léger. Chaque fonctionnalité est un agent indépendant.

  • FastAPI Core — Orchestrateur central exposé sur :8000. Point d'entrée unique.

  • Agents modulaires — LLM, STT, TTS, Mémoire, Telegram s'activent selon la config.

  • Ollama natif — Communication directe HTTP avec Ollama local. Aucun intermédiaire.

  • SQLite embarqué — Base de données locale, zéro dépendance externe.
